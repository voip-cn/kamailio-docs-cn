1. Overview

This module provides limited STUN server (RFC 5389) support for Kamailio. It's built for providing STUN support in the SIP signalling channel, not being a standalone STUN/TURN server.

2. Dependencies

2.1. Kamailio Modules

The following modules must be loaded before this module:

  • None

2.2. External Libraries or Applications

The following libraries must be installed before running Kamailio with this module loaded:

  • none.
