1. Overview

This is an example module. It implements only one function that prints its string parameter to stderr (it won't work if server is started in daemon mode). Its prupose is to show how to link to an internal library (lib/print).

2. Functions

2.1.  print_stderr(txt)

Prints string to stderr.

Meaning of the parameters is as follows:

  • txt - string to be printed.
